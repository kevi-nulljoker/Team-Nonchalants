# 💰 FinSight

FinSight is a full-stack personal expense tracking web application that helps users monitor spending, classify transactions intelligently, and gain financial insights.

---

## 🚀 Features

- 📊 Interactive expense dashboard
- 🤖 ML-based expense classification
- 📂 JSON transaction import
- 🔐 Secure backend API
- ⚡ Fast React frontend
- 📈 Clean and modern UI

---

## 🛠 Tech Stack

### Frontend
- React
- Vite
- JavaScript
- CSS

### Backend
- Python
- Flask
- Scikit-learn
